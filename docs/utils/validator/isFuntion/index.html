isFunction

주어진 인자가 함수인지 검사하고, 맞다면 인자의 타입을 Function로 좁혀주는 함수입니다.


Interface​

typescript
const isFunction: (arg: unknown) => arg is Function

Usage​

import { isFunction } from '@modern-kit/utils';

function example() {}

isFunction(() => {}); // true
isFunction(example); // true

isFunction('123'); // false
isFunction(123); // false
isFunction({}); // false
isFunction([]); // false
