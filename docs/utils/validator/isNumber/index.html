isNumber

주어진 인자가 숫자인지 검사하고, 맞다면 인자의 타입을 number로 좁혀주는 함수입니다.


Code​

🔗 실제 구현 코드 확인

Interface​

typescript
const isNumber: (arg: unknown) => arg is number

Usage​

typescript
import { isNumber } from '@modern-kit/utils';


isNumber(123); // true

isNumber(() => {}); // false
isNumber('123'); // false
isNumber(true); // false
isNumber({}); // false
isNumber([]); // false
