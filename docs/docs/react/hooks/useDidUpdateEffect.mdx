# useDidUpdateEffect

최초 마운트 시에는 실행되지 않고, 의존성 배열(`deps`)의 값이 업데이트되었을 때만 effect를 실행시키는 커스텀 훅입니다.

<br />

## Code
[🔗 실제 구현 코드 확인](https://github.com/modern-agile-team/modern-kit/blob/main/packages/react/src/hooks/useDidUpdateEffect/index.ts)

## Interface
```ts title="typescript"
function useDidUpdateEffect(effect: EffectCallback, deps: DependencyList): void
```

## Usage
```tsx title="typescript"
import { useDidUpdateEffect } from '@modern-kit/react'

const Example = () => {
  const [count, setCount] = useState(0);
  const [isUpdated, setIsUpdated] = useState(false);

  useDidUpdateEffect(() => {
    setIsUpdated(true);
  }, [count]);


  return (
    <div>
      <button onClick={() => setCount(count + 1)}>Increment: {count}</button>
      {isUpdated && <p>count가 변경되었습니다. {count}</p>}
    </div>
  );
}
```
