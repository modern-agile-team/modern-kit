# useWindowScrollTo

`window.scrollTo()`를 편하게 호출하는 커스텀 훅입니다.

<br />

## Code
[🔗 실제 구현 코드 확인](https://github.com/modern-agile-team/modern-kit/blob/main/packages/react/src/hooks/useWindowScrollTo/index.ts)

## Interface
```ts title="typescript"
interface UseScrollToProps {
  left?: ScrollToOptions['left']; // default: 0
  top?: ScrollToOptions['top']; // default: 0
  behavior?: ScrollToOptions['behavior']; // default: 'auto'
  dependencies?: DependencyList; // default: []
}

const useWindowScrollTo: ({
  left,
  top,
  behavior,
  dependencies,
}?: UseScrollToProps) => void;
```

## Usage
```tsx title="typescript"
import { useWindowScrollTo } from '@modern-kit/react';

const Example = () => {
  useWindowScrollTo();

  return (<div>{/* ... */}</div>);
};
```
