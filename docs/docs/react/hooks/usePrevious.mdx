# usePrevious

이전 렌더링에서 인수의 값을 반환하는 커스텀 훅입니다.

<br />

## Code
[🔗 실제 구현 코드 확인](https://github.com/modern-agile-team/modern-kit/blob/main/packages/react/src/hooks/usePrevious/index.ts)

## Interface
```ts title="typescript"
const usePrevious: <T>(value: T) => T;
```

## Usage
```tsx title="typescript"
import { useState } from 'react';
import { usePrevious } from '@modern-kit/react';

const Example = () => {
  const [count, setCount] = useState(0);
  const previousCount = usePrevious(count);

  const onIncrementCount = () => {
    setCount(count + 1);
  };

  return (
    <div>
      <div>Current Count is - {count}</div>
      <div>Previous Count is - {previousCount}</div>
      <button onClick={onIncrementCount}>Increment</button>
    </div>
  );
};
```
