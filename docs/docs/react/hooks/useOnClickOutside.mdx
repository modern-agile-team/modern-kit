# useOnClickOutside

`ref`를 할당한 타겟 엘리먼트 외부 요소를 클릭 할 경우 콜백 함수를 호출하는 커스텀 훅입니다.

<br />

## Code
[🔗 실제 구현 코드 확인](https://github.com/modern-agile-team/modern-kit/blob/main/packages/react/src/hooks/useOnClickOutside/index.ts)

## Interface

```ts title="typescript"
const useOnClickOutside: <T extends HTMLElement>(
  action: (targetElement: T) => void
) => {
  ref: React.RefObject<T>;
};
```

## Usage
```tsx title="typescript"
import { useMemo } from 'react';
import { useOnClickOutside } from '@modern-kit/react';

const Example = () => {
  const { ref } = useOnClickOutside<HTMLDivElement>(() => {
    window.alert('outside click')
  });
  
  const boxStyle = useMemo(() => {
    return {
      width: '400px',
      height: '400px',
      background: '#439966',
      fontSize: '1.5rem',
      color: '#fff',
    };
  }, []);

  return (
    <div ref={ref} style={boxStyle}>
      Target Box
    </div>
  );
};
```
