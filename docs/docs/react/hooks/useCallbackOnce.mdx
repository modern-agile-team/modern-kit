# useCallbackOnce

`useCallbackOnce`는 주어진 콜백 함수가 최초 실행 이후 다시 실행되지 않도록 보장하는 커스텀 훅입니다. 
 의존성 배열(`deps`)이 변경될 때마다 콜백을 새로 메모이제이션합니다.

<br />

## Code
[🔗 실제 구현 코드 확인](https://github.com/modern-agile-team/modern-kit/blob/main/packages/react/src/hooks/useCallbackOnce/index.ts)

## Interface
```ts title="typescript"
function useCallbackOnce<F extends (...args: any[]) => void>(callback: F, deps: DependencyList): F;
```

## Usage
```tsx title="typescript"
import { useCallbackOnce } from '@modern-kit/react';

const Example = () => {
  const handleClick = useCallbackOnce((e: React.MouseEvent<HTMLDivElement, MouseEvent>) => {
    console.log('최초 한번만 실행됩니다.');
  }, []);

  return <div onClick={handleClick}>Click me</div>;
};
```

## Example
