# Delay

지정된 시간이 지난 후에 `자식 컴포넌트`를 렌더링하는 컴포넌트입니다.

지연 시간 동안에는 `fallback` 컴포넌트가 렌더링됩니다.

<br />

## Code
[🔗 실제 구현 코드 확인](https://github.com/modern-agile-team/modern-kit/blob/main/packages/react/src/components/Delay/index.tsx)

## Interface
```ts title="typescript"
interface DelayProps {
  children: React.ReactNode;
  delay: number;
  fallback?: React.ReactNode;
}
```
```tsx title="typescript"
const Delay: ({ children, delay, fallback }: DelayProps) => JSX.Element;
```

## Usage
```tsx title="typescript"
import { Delay } from '@modern-kit/react'

const Example = () => {
  return (
    <Delay
      delay={2000}
      fallback={
        <div>
          fallback입니다. 2초 뒤에 children이 렌더링됩니다.
        </div>
      }>
      <div>children입니다.</div>
    </Delay>;
  );
};
```
