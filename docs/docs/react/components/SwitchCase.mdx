# SwitchCase

`Switch` 문을 컴포넌트 형태로 사용할 수 있는 컴포넌트입니다.

`caseBy`에는 `key(condition):value(component)` 형태의 프로퍼티를 갖는 객체를 넘겨주고, `value` props에 매칭되는 `value(컴포넌트)`를 렌더링합니다.

`caseBy`에 매칭되는 `value`가 없을 때 임시로 보여줄 수 있는 `defaultComponent`를 `props`로 넘겨줄 수 있습니다.

## Interface
```ts title="typescript"
interface SwitchCaseProps<Case extends PropertyKey> {
  value: Case | null | undefined;
  caseBy: Record<Case, React.ReactNode>;
  defaultComponent?: React.ReactNode;
}
```
```ts title="typescript"
const SwitchCase: <Case extends PropertyKey>({
  value,
  caseBy,
  defaultComponent,
}: SwitchCaseProps<Case>) => JSX.Element;
```

<br />

### Props

| Name | Type | Default | Description |
|------|------|---------|-------------|
| `value` | `Case \| null \| undefined` | - | 매칭할 값 |
| `caseBy` | `Record<Case, React.ReactNode>` | - | 조건(key)과 렌더링할 컴포넌트(value)의 매핑 객체 |
| `defaultComponent` | `React.ReactNode` | `undefined` | 매칭되는 case가 없을 때 렌더링할 컴포넌트 |

## Usage
```tsx title="typescript"
import { SwitchCase } from '@modern-kit/react';


const Example = () => {
  const [condition, setCondition] = useState(1);

  return (
    <div>
      <div>
        <button onClick={() => setCondition(1)}>Change to case 1</button>
        <button onClick={() => setCondition(2)}>Change to case 2</button>
        <button onClick={() => setCondition(3)}>Change to case 3</button>
      </div>

      <SwitchCase
        value={condition}
        caseBy={{
          1: <h3>Case No.1</h3>,
          2: <h3>Case No.2</h3>,
          3: <h3>Case No.3</h3>,
        }}
      />
    </div>
  );
};
```
