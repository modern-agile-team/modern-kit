Portal

Portal 컴포넌트는 React.createPortal을 사용해 부모 컴포넌트의 DOM 계층 구조 외부의 DOM 노드에서 자식 컴포넌트를 렌더링합니다.

Portal 컴포넌트는 기본적으로 document.body에 자식 컴포넌트를 렌더링합니다. 하지만 Portal 컴포넌트의 containerRef 프로퍼티를 전달하면 자식 컴포넌트를 document.body가 아닌 다른 DOM Element에서 렌더링할 수 있습니다.

또한, 중첩 포털 기능을 지원합니다. 여러 Portal 컴포넌트를 중첩하면 중첩된 Portal DOM 계층 구조가 생성됩니다.

Portal 컴포넌트는 Modal, Dialog, Tooltip과 같은 기능과 함께 사용하기에 이상적입니다.

React CreatePortal 다음 문서를 참고해주세요.


Code​

🔗 실제 구현 코드 확인


Interface​

typescript
function Portal({
  children,
  containerRef,
  className,
}: {
  children: React.ReactNode;
  containerRef?: React.RefObject<HTMLElement | null>;
  className?: string;
}): JSX.Element;

Props​

NameTypeDefaultDescription
childrenReact.ReactNode-포털로 렌더링할 자식 컴포넌트
containerRefReact.RefObject<HTMLElement | null>document.body자식을 렌더링할 DOM 컨테이너 ref
classNamestringportal포털 컴포넌트에 적용할 클래스 이름

Usage​

기본 사용법​

typescript
import { Portal } from '@modern-kit/react'

const Example = () => {
  return (
    <Portal>
      <p>Example Portal</p>
    </Portal>
  );
};

containerRef 속성 사용법​

typescript
import { Portal } from '@modern-kit/react'

const Example = () => {
  const ref = useRef<HTMLDivElement | null>(null);

  return (
    <div>
      <Portal containerRef={ref}>
        <p>Example Portal</p>
      </Portal>

      <div id="outer" ref={ref} />
    </div>
  );
};

중첩 Portal 사용법​

typescript
import { Portal } from '@modern-kit/react'

const Example = () => {
  return (
    <Portal>
      <p>Default Portal</p>
      <Portal>
        <p>Nested Portal1</p>
        <Portal>
          <p>Nested Portal2</p>
        </Portal>
      </Portal>
    </Portal>
  );
};
