# @modern-kit/docs

The folder that manages official documents for `@modern-kit`.

<br />

## Local Start
1. Create a `.md` file in the `prepareDocs` folder.

2. Execute the commands below in sequence.
```shell
yarn prepare:docs
```

```shell
yarn start
```
