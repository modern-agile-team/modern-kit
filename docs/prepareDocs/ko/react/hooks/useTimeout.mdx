# useTimeout

`window.setTimeout`을 편리하게 사용할 수 있는 커스텀 훅 입니다. 

`delay`의 값으로 `null`을 넣으면 타이머를 중지할 수 있습니다.

<br />

## Interface
```tsx
type SetTimeoutParameters = Parameters<typeof setTimeout>;

const useTimeout: (callback: SetTimeoutParameters[0], delay?: SetTimeoutParameters[1]) => void
```

## Usage

```tsx
import { useTimeout } from '@modern-kit/react';

const Example = () => {
  const [number, setNumber] = useState(0);

  useTimeout(() => {
    setNumber(number + 1);
  }, 1000);

  useTimeout(() => {
    setNumber(number + 1);
  }, 2000);

  return <div>{number}</div>;
};
```
