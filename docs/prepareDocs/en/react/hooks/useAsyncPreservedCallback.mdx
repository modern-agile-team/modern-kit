# useAsyncPreservedCallback

A custom hook that is similar to [usePreservedCallback](https://team-grace.github.io/devgrace/docs/react/hooks/usePreservedCallback/) but returns a Promise.

<br />

## Interface
```tsx
const useAsyncPreservedCallback: <T extends (...args: any[]) => Promise<any>>(
  callback: T
) => (...args: any[]) => Promise<any>;
```
